The HMC5883 is a magnetometer using [I2C](../I2C-README.md). The Arduino library is available from the Library Manager as Adafruit HMC5883 Unified.

## Pins:
Arduino    | Sensor
-----------|-------
  GND      | GND
  5V       | VIN
  SCL/A5/3 | SCL
  SDA/A4/2 | SDA

The example code is available in hmc5883-example.ino.
